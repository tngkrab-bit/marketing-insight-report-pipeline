# Agent Guidance

This repository is intended for public, deterministic marketing analytics examples.

- Do not include confidential, customer, client, or personally identifiable data.
- Never fabricate campaign results or KPI values.
- Calculate performance metrics deterministically in Python before generating any AI-written interpretation.
- Require tests for changes to validation, metric calculation, ranking, or reporting logic.
- Treat API key exposure and confidential data leakage as critical issues.
- Keep README content, code comments, and documentation in English.
- Use synthetic or public-safe sample data only.
